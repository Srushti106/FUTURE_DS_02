## Future Interns - Data Science & Analytics Internship

## Objective
Analyze customer data to identify churn patterns, 
key retention drivers, and customer lifetime trends.

## Tools Used
- Power BI Desktop
- Telco Customer Churn Dataset (Kaggle)
- GitHub

## Dataset
Source: Kaggle - Telco Customer Churn
Records: 7,043 customers

## Key Insights
- Month-to-month contracts churn the most
- Fiber optic users have highest churn rate
- Electronic check payment shows most churn
- New customers with low tenure churn more
- Overall Churn Rate: 26.54%

- Retained Customers: 5,174
